1. Целое число - это число, которое можно записать без дроби.


2. l-values - наличие у значения собственного адреса в памяти.


3. r-values - не наличие у значения постоянного адреса в памяти.


4. Инициализация - присвоение значения только что объявленной переменной.


5. Неинициализированные - объявленные переменные без определённого значения. 


6. Присваивание - дать новое значение переменной.


7. << - оператор вывода 


8. >> - оператор ввода


9. Функция - это последовательность стэйтментов для выполнения определённого задания 


10. Список флагов в (Other compiler options) для контроля написания кода 
    -Wall
    -Wextra
    -Werror
    -pedantic
    -pedantic-errors
    -Waddress
    -Wconversion
    -Wshadow
    -Wfloat-equal
    -Wuninitialized
    -Wundef
    -Wpointer-arith
    -Wcast-align
    -Wlogical-op
    -Wmissing-declarations
    -Wredundant-decls

    Писать флаг -Wfatal-errors только в том случае, когда начинается писанина тяжеловесного проекта (не учебного)

    
11. caller - вызывающая функция
